# ROS Sphero Bolts

`demiurge-sphero-bolt` is a ROS meta-package to operate a swarm of Sphero Bolt robots.

### Requirements

Ubuntu 20.04 Focal Fossa

ROS Noetic

Python libraries: [`spherov2.py`](https://github.com/poiuytrezaur/spherov2.py.git), bleak
